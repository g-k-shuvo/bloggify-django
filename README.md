# Bloggify
> Full Featured blog application using Django Web Framework

### Features:
- Blog List
- Blog Details
- Searching Option
- Categories
- Category Wise Posts
- Featured Posts
- User Profile Create, Update, Delete
- User Post Create, Update, Delete
- Post Like Comments
- Admin  Panel
- Login / Logout
- and Many More...
